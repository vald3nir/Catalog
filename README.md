# Catalog API

Example of API Rest of an Items Catalog with .NET 5/C#

## Project features

- Clean architecture
- Exploring the default Swagger UI page
- Persisting Entities with MongoDB
- Use dependency injection
- Use Tasks, Async and Await
- Secrets and Health Checks
- Docker
